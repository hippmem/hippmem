# hippmem-consolidation

Consolidation engine for HIPPMEM — Hebbian reinforcement, decay, compaction, summary merge, contradiction detection, and the background worker.

Part of [HIPPMEM](https://github.com/hippmem/hippmem) — a native associative memory engine for AI agents, written in Rust.
