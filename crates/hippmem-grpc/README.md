# hippmem-grpc

HIPPMEM gRPC bindings — protobuf mapping for core types and a tonic service, feature-gated under the `grpc` feature.

Part of [HIPPMEM](https://github.com/hippmem/hippmem) — a native associative memory engine for AI agents, written in Rust.
