# hippmem-retrieval

Retrieval engine for HIPPMEM — multi-channel seed recall (BM25 + entity + semantic + temporal + topic + graph), RRF fusion, spreading activation, rerank, and explanation traces.

Part of [HIPPMEM](https://github.com/hippmem/hippmem) — a native associative memory engine for AI agents, written in Rust.
