# hippmem-store

Storage and indexing for HIPPMEM — redb persistence, full-text index (Tantivy), HNSW vector index, inverted indexes, and the association graph store.

Part of [HIPPMEM](https://github.com/hippmem/hippmem) — a native associative memory engine for AI agents, written in Rust.
